# Financial Crime Quality Assurance Lab

## Overview

This project simulates an AML (Anti-Money Laundering) Quality Control environment using a synthetic dataset.

The dashboard demonstrates how financial crime teams can monitor suspicious activity, review investigation outcomes, analyse quality control results, and track Suspicious Matter Report (SMR) recommendations.

## Tools Used

- Power BI
- Excel
- DAX
- AML Analytics

## Dashboard Pages

### Executive Overview

- Total Transactions
- Total Alerts
- High Risk Customers
- SMR Cases
- QC Fail Rate
- Risk Distribution
- Transaction Activity by Country

### AML Investigation Centre

- Investigator Decision Analysis
- QC Failure Analysis
- SMR Analysis by Alert Type
- Case Review Details

## Key Features

- AML Alert Monitoring
- Customer Risk Scoring
- Investigation Review Tracking
- Quality Control Analysis
- SMR Monitoring
- Interactive Power BI Dashboards

## Disclaimer

This project uses a simulated AML dataset created for learning and portfolio purposes only. No real customer or financial institution data was used.
